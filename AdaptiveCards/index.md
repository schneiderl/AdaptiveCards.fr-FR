## <a name="how-they-work"></a><span data-ttu-id="cf560-104">Fonctionnement</span><span class="sxs-lookup"><span data-stu-id="cf560-104">How they work</span></span>

## <a name="core-design-principles"></a><span data-ttu-id="cf560-136">Principes de conception de base</span><span class="sxs-lookup"><span data-stu-id="cf560-136">Core Design Principles</span></span>

### <a name="semantic-instead-of-pixel-perfect"></a><span data-ttu-id="cf560-138">Privilégier la sémantique à une disposition de type « pixel parfait »</span><span class="sxs-lookup"><span data-stu-id="cf560-138">Semantic instead of pixel-perfect</span></span>
